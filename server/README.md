# Chinese - Vietnamese Ocr Aligning Tool

## How to run

### Client side: React

Go into client's directory

```sh
cd client
```

Install dependencies

```sh
npm install
```

Run the client

```sh
npm run dev
```

### Server side: Flask

Go into server's directory

```sh
cd server
```

Install requirements

```sh
pip install -r requirements.txt
```

Run the server

```sh
flask run --debug
```
